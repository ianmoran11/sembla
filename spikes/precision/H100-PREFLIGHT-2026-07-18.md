## Environment

- Provider: Hyperstack, `CANADA-1`
- Flavor: `n3-H100x1`
- GPU: one NVIDIA H100 PCIe, classified by the spike as full-rate FP64 (`1:2`)
- Image: Ubuntu Server 22.04 LTS R570 CUDA 12.8
- Full workload: 26,000,000 rows and 1,300,000 groups
- Benchmark protocol reached: 10 warmup ticks plus 100 measured ticks per available strategy

## Diagnostic findings

1. CUDA 12.8 compilation requires `<math_constants.h>` for `CUDART_INF`.
2. The pinned Rust 1.77 toolchain requires an `extern "C"` block rather than an
   `unsafe extern "C"` block.
3. Native CUDA `f64` passed its one-million-row accuracy guard.
4. `f32` passed its accuracy guard.
5. Double-single failed its guard on NVIDIA/Vulkan because its winner mismatch
   rate (`0.00002`) did not improve on the `f32` baseline (`0.00002`). The
   backend also did not provide the strict-math guarantees required to qualify
   double-single.
6. The wgpu 0.20 native-`f64` shader reached NVIDIA's Vulkan compiler, which
   reported `NVVM compilation failed`. Even after the validation error was
   captured, the process segfaulted during device teardown. The tested
   `NVIDIA H100 PCIe` adapter is therefore gated before pipeline creation, while
   the independent CUDA path remains available.
7. After that gate, all 32 release tests passed on the H100 and the full CUDA
   benchmark invocation completed successfully.

Observed guard summary:

```text
accuracy guard [double-single]: failed
accuracy guard [f32]: passed
accuracy guard [native f64 (CUDA)]: passed
accuracy guard [native f64 (wgpu)]: unavailable (observed H100 NVVM compiler failure)
```

## Provisional roadmap implications

- Strategy A (native `f64`) is technically viable through CUDA on the tested H100.
- Strategy B (double-single) must not be assumed portable across Metal and
  NVIDIA/Vulkan; it was unqualified in this preflight.
- The likely NVIDIA production route is CUDA native `f64` with the CPU `f64`
  oracle retained for differential validation.
- wgpu native `f64` remains potentially testable on other Vulkan adapters, but
  must not be used on the observed H100 configuration.
- These are feasibility findings, not the final ADR selection.
